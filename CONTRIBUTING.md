# Contribuer au projet

Merci de vouloir améliorer ce cours ! Voici comment faire :

1. Forkez le projet.
2. Créez votre branche ().
3. Committez vos changements ().
4. Pushez sur la branche ().
5. Ouvrez une Pull Request.
